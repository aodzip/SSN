湿湿娘主站项目
目录结构
    ├── css       
    ├── fonts
    ├── images
    ├── js
    ├── less
    ├── mycss
    ├── mydoc
    ├── myimages
    ├── myjs
目录结构说明
    css,fonts,images,js,less    目录为musik-music模板库的资源目录
    mycss,mydoc,myimages,myjs   目录为自定义目录,存储自己的代码
    
    .html文件统一放置在根目录下如
        demo.html(模板演示)
        template.html(开发基础模板)
框架分析
    CSS框架
        jPlayer Version 2.6.0
        Bootstrap v3.1.1
        Font Awesome 4.0.3(字体图标库)
        animate(CSS3动画库)
        simple-line-icons(线条图标库)
    JS框架
        jQuery v1.10.2
        Bootstrap v3.1.1
        jquery.slimscroll v1.3.0
        jPlayer Version 2.6.0
注意
    1.理论上musik-music模板库提供的资源已经能够满足我们的开发需求,所有的资源都直接从模板库中寻找,如需引人新的框架,请在团队讨论申请
    2.请不要修改模板库的任何代码,如需修改请使用覆盖的方式,在自定义代码文件夹中写代码覆盖之
    3.模板库兼容到IE9
